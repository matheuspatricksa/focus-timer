# Focus Timer

Um aplicativo de timer básico focado em ajudar você a gerenciar seus períodos de trabalho e descanso, baseado na técnica Pomodoro.

## Funcionalidades

- Temporizador de sessões de foco e pausas
- Alertas sonoros ao final de cada ciclo
- Interface simples e intuitiva
- Som de fundo

## Como usar

1. Defina o tempo desejado para foco e pausa.
2. Clique no botão "Play" para começar o timer.
3. Ao final do ciclo, siga o alerta para descansar ou voltar ao trabalho.

## Instalação

```bash
git clone https://github.com/matheuspatricksa/focus-timer.git
cd focus-timer
# Siga as instruções de instalação conforme a tecnologia do projeto
```
